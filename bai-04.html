Câu 2: Bài tập về background

Yêu cầu 1: Tạo 1 thẻ div, đặt tên class cho thẻ này tùy ý, sau đó viết css theo tên class đã đặt. Chiều rộng 200px, chiều cao 100px. Có background màu xanh lá.

Yêu cầu 2: Tạo 1 thẻ div đặt thêm class cho thẻ này tùy ý, sau đó viết css theo tên class đã đặt. Chiều rộng 100% chiều cao 200px. Có background là hình ảnh, khi scroll ảnh đứng yên.
